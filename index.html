Elevia: E-Commerce

Una plataforma de comercio electrónico diseñada para ofrecer una experiencia de compra fluida y accesible en cualquier dispositivo. Cuenta con un catálogo interactivo, carrito de compras dinámico y autenticación segura para garantizar una navegación intuitiva y confiable.

Características

  • Responsividad total – Diseño adaptable a cualquier dispositivo para una mejor experiencia de usuario.
  • Seguridad y eficiencia – Autenticación con JWT y manejo estructurado de datos con DTOs y MySQL.
  • Arquitectura escalable – Desarrollo modular con APIs RESTful y control de versiones con GIT.

Reddit Minimal

RedditMinimal es una interfaz minimalista de Reddit desarrollada con React, JavaScript, HTML y CSS. Se alimenta de la API externa de Reddit para mostrar publicaciones de diferentes categorías, sus comentarios respectivos y una barra de búsqueda para facilitar la navegación.

Características

  • Interfaz simple y minimalista inspirada en Reddit.
  • Obtención dinámica de publicaciones de diversas categorías mediante la API de Reddit.
  • Visualización de comentarios dentro de cada publicación
  • Barra de búsqueda para navegar por las publicaciones.
